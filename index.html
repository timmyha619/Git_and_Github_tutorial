Overview of Git and GitHub

Git

Git, created by Linus Torvalds, is a system that track changes of files, which allows coordinated work between programmers. Since Gitâ€™s design aids in branching and merging, non-linear workflow is incredibly supported, so developers can each work in their own portion of the project and later merged to create a full branch structure. Git can also allow programmers to work on the same code without any complication. Git works as a version control system that allows programmers to track revision of a code, when was it changed and who execute the change, revert to the specific versions if needed, and prevent accidental overwrites of older file.

GitHub

GitHub is a platform that allows users to store their projects in repository, and to connect to people, usually who work in the same field as theirs. With repository, programmers can organize and store their files for the project in a single repo, with a unique URL created for each repo. With the feature of repository, users can fork a repo, or revise an existing project. If a developer found an interesting project from another developer, he/she may fork the repo, revise it, and save it as a new repository. Additionally, if you want to your revision to be acknowledged by the original developer, or even have your work included in the final project, you can create a pull request, so that the author can view your contribution. And all of this is possible thanks to the social networking aspect of GitHub. GitHub allows users to explore projects from other authors, and even contribute to these projects. GitHub also saves your progress and contributions of yours and others projects as a profile, so you can use it as your resume for your job application.

How to install Git and GitHub?

To install Git, you can go to https://git-scm.com/downloads. Then, you need to choose the proper version according to the platform of your device. For GitHub, you have the options of either accessing it via https://github.com/, or download the application at https://desktop.github.com/.
